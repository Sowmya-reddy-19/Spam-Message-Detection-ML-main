\# Spam Message Detection Using Machine Learning



\## Project Overview



This project detects whether an SMS message is Spam or Ham (Not Spam) using Machine Learning techniques.



\## Technologies Used



\* Python

\* Pandas

\* Scikit-Learn

\* Matplotlib



\## Algorithms Used



1\. Multinomial Naive Bayes

2\. Logistic Regression



\## Features



\* Data Preprocessing

\* Text Vectorization using CountVectorizer

\* Train-Test Split

\* Cross Validation

\* ROC-AUC Score

\* Algorithm Comparison

\* Spam/Ham Prediction

\* Pie Chart Visualization



\## Results



\* Naive Bayes Accuracy: 97.85%

\* Logistic Regression Accuracy: 97.76%

\* ROC-AUC Score: 0.9792

\* Average Cross Validation Score: 0.9808
